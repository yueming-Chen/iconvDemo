程式需求：node.js ^6.3.1
請把資料夾放在C:/Users/"你的名字"/Desktop/a/
進到API.js更改所有C:/Users/"你的名字"/Desktop/a  為你的目的地
選擇sinfofrom.data

/******************/

1.啟動command line，cd進此資料夾
2.輸入npm install
3.輸入npm start
4.打開browser(chrome , firefox....etc)，輸入localhost:3067  /*Port已cmd出現的port號為主*/
5.選擇的檔案必須放在此目錄下，點選後按下submit即可解析資料，點選write會把解析資料寫進output.txt
